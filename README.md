# Password Generator

A simple, secure, and customizable password generator application. Easily create strong passwords for your accounts and keep your online presence safe!

## Features

- Generate random passwords of variable length
- Choose between different character sets (uppercase, lowercase, numbers, symbols)
- Copy passwords to clipboard with one click
- User-friendly interface
- No data stored or shared

## Installation

Clone this repository:

```bash
git clone https://github.com/MohammedGousemohiyuddinYekkaluru/Password-Generator.git
cd Password-Generator
```

## Configuration

- Set password length
- Toggle inclusion of uppercase, lowercase, numbers, and symbols
